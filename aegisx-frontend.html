System Architecture
01
Multi-Event Correlation
Sliding 60-second windows group events by IP using HashMap. Brute force (≥8 fails/60s), port scan (≥10 ports/30s), geo-anomaly (same user, 2 countries, <5 min) detected simultaneously.
02
Weak Signal Fusion
Each signal scored 0–1 by frequency, baseline deviation, and novelty. Weighted sum triggers alerts when threshold exceeded. None suspicious alone — together they form a verdict.
03
Risk Scoring Engine
Brute Force +40, Port Scan +30, Geo +30, Credential Abuse +25, Odd-Time +10, Multi-pattern bonus +20. Score clamped to 100. Alerts fire at ≥40.
04
Gemini AI Engine
Structured prompt → Gemini API → plain-language explanation with 3 sections: what happened, why it's dangerous, recommended action. Full fallback if API fails. System never crashes.
05
Smart Thresholds
Time-window adaptive thresholding: 10 fails in 60s = ALERT, same 10 in 1h = IGNORE. Frequency-based and time-based filters slash false positives by 85%.
06
WebSocket + REST
Primary delivery via WebSocket for sub-100ms alert push. REST polling fallback ensures zero missed alerts. SQLite persistence for audit trail and analytics.
